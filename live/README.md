# Live

`live` 是 `craw` 仓库下用于沉淀飞书 Open Craw 直播相关专项信息的目录。

## 本专项关注内容

- 直播相关业务背景
- 直播场景下的能力需求和问题拆解
- 直播专项调研记录
- 直播专项方案决策
- 当前推进事项

## 当前结构

- [`overview.md`](./overview.md)：专项背景、目标与边界。
- [`todos.md`](./todos.md)：待办和推进项。
- [`lazy-pack-onboarding/`](./lazy-pack-onboarding/)：飞书懒人包用户的 Open Claw 7 天上手服务方案与工程文档。
- [`notes/`](./notes/)：会议记录、调研笔记。
- [`decisions/`](./decisions/)：关键方案与结论。
- [`references/`](./references/)：资料索引和外部引用。
